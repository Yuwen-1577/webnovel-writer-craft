> 网文实战套路模板（打脸爽文/废材逆袭/扮猪吃虎/类型化框架）详见 `plan/references/outlining/plot-frameworks.md`。

## 一、六大框架速查表

| 框架 | 核心思想 | 节拍数 | 最适合题材 | 网文适配度 |
|------|---------|--------|-----------|:--------:|
| **三幕式** | 设置→对抗→解决 | 3 幕 | 通用 | ★★★★★ |
| **英雄之旅** | 心理转变的骨架 | 12 阶段 | 玄幻/修仙/冒险 | ★★★★ |

## 二、Save the Cat 15 节拍与网文章节映射

Save the Cat（Blake Snyder）是**最适合网文章节规划的西方框架**。Jessica Brody 的《Save the Cat! 写小说》将其从小说角度重新诠释。

| 节拍 | 位置 | 网文对应 | 作用 |
|------|------|---------|------|
| 开场意象 | 0-1% | 第 1 章开头 | 一个画面定下基调 |
| 主题陈述 | 5% | 第 1 章中段 | 暗示深层主题 |
| 铺垫 | 1-10% | 黄金三章 | 介绍主角、世界、缺陷 |
| **催化事件** | **10%** | **第 3 章左右** | **打破日常的触发点** |
| 犹豫 | 10-20% | 第 4-6 章 | 主角内心挣扎 |
| **跨入第二幕** | **20%** | **第 6 章左右** | **主角做出选择** |
| B 故事线 | 22% | 感情线/导师线启动 | 通常是爱情线或师徒线 |
| 趣味博弈 | 20-50% | 升级打怪段 | 故事"承诺的兑现"——核心卖点 |
| **中点** | **50%** | **卷中** | **假胜利或假失败** |
| 坏人逼近 | 50-75% | 反派施压段 | 外部压力+内部缺陷双线收紧 |
| **全盘皆输** | **75%** | **卷末前** | **最低点，常伴随"死亡的气息"** |
| 灵魂暗夜 | 75-80% | 低谷反思 | 主角消化失败，发现真相 |
| 跨入第三幕 | 80% | 转折点 | 新灵感催生最终方案 |
| 终局 | 80-99% | 最终对决 | 正面对抗，完成转变 |
| 终场意象 | 99-100% | 卷末 | 与开场意象镜像，展示变化 |

### 网文卷级应用

一卷（约 30-50 章）可以映射一次完整 STC 节拍：

```
第 1-3 章：铺垫 + 催化事件（黄金三章）
第 4-10 章：犹豫 + 跨入第二幕 + 趣味博弈
第 11-20 章：中点（假胜利/假失败）+ 坏人逼近
第 21-25 章：全盘皆输 + 灵魂暗夜
第 26-30 章：终局 + 终场意象 + 新钩子
```

## 三、英雄之旅 12 阶段网文化

Campbell/Vogler 的英雄之旅是**心理转变的骨架**，不是情节模板。

| 阶段 | 原名 | 网文常见表现 |
|------|------|------------|
| 1. 平凡世界 | Ordinary World | 主角是废材/普通人 |
| 2. 冒险召唤 | Call to Adventure | 遭遇危机（退婚/亲人被杀/世界末日） |
| 3. 拒绝召唤 | Refusal | 一开始不想卷入 |
| 4. 遇见导师 | Meeting the Mentor | 获得金手指/师尊指点 |
| 5. 跨越门槛 | Crossing the Threshold | 加入宗门/进入修仙界 |
| 6. 试炼与盟友 | Tests, Allies, Enemies | 遇到死党、女主，经历小战斗 |
| 7. 深入洞穴 | Approach to the Inmost Cave | 进入危险区域（秘境/敌对势力） |
| 8. 苦难考验 | The Ordeal | 遭遇最大危机 |
| 9. 奖赏 | Reward | 获得神器/顿悟/突破境界 |
| 10. 返回之路 | The Road Back | 带着收获回归 |
| 11. 复活 | Resurrection | 浴火重生，实力大涨 |
| 12. 携万灵丹归来 | Return with the Elixir | 成为传奇，改变世界 |

**Vogler 七大角色原型**：英雄、信使、变形者、导师、阴影（反派）、骗术师、守门人。

## 四、起承转合 — 非冲突驱动的东方叙事

| 段 | 功能 | 关键差异 |
|----|------|---------|
| 起 | 引入人物、情境 | 同三幕式 |
| 承 | 深化前提，自然发展 | 同 Rising Action |
| **转** | **引入意外元素** | **不一定是冲突——可以是发现、感悟、视角转换** |
| 合 | 调和意外与前文 | 同 Resolution |

**适用场景**：都市日常、治愈系、情感线、心理描写。不是所有叙事都需要反派。

## 五、框架选择决策树

```
你的题材是？
├── 玄幻/修仙/冒险
│   ├── 读者是小白 → 英雄之旅 + 打脸爽文循环
│   └── 读者是老白 → 三幕式 + 英雄之旅深层结构
├── 都市/现代
│   ├── 异能/超自然 → Save the Cat + 扮猪吃虎
│   └── 日常/情感 → 起承转合（不强制冲突）
├── 言情
│   ├── 甜宠 → Save the Cat（B 故事线=感情线）
│   └── 虐恋 → 三幕式 + 负向角色弧光
├── 悬疑/推理
│   └── Freytag 金字塔（张力精确控制）
└── 无限流/副本
    └── 副本内 STC 节拍 + 主线英雄之旅
```

### 组合使用

单一框架容易单调，常见组合：
- **英雄之旅 + 打脸爽文**：结构用英雄之旅，节奏用打脸循环
- **Save the Cat + 起承转合**：主线用 STC，情感线用起承转合
- **三幕式 + 雪花法**：架构用三幕式，设计过程用雪花法的分形扩展

## 六、网文模式与叙事理论的映射

网文特有模式不是"低级套路"，它们是经典叙事理论的网文化改造。以下是精确映射：

### 打脸爽文 = McKee 期望鸿沟的循环应用

| 打脸节拍 | McKee 对应 | STC 节拍 |
|---------|-----------|---------|
| 被嘲讽 | 设定期望（读者知道主角被低估） | 铺垫 |
| 隐忍 | 期望延迟（读者等待爆发） | 趣味博弈 |
| 爆发 | 期望鸿沟（读者预期被超越） | 中点/终局 |
| 打脸 | 价值转变（受辱→受敬） | — |
| 新嘲讽者出现 | 新一轮期望设定期望 | 坏人逼近 |

**核心**：打脸不是"主角赢了"，是"读者的期待被超越了"——McKee 的期望鸿沟。

### 废材逆袭 = 英雄之旅的压缩版

| 废材逆袭阶段 | 英雄之旅对应 | STC 对应 |
|------------|------------|---------|
| 废材期（被欺负） | 平凡世界 + 冒险召唤 | 铺垫 + 催化事件 |
| 奇遇期（获得金手指） | 遇见导师 + 跨越门槛 | 跨入第二幕 |
| 崛起期（打脸升级） | 试炼与盟友 + 深入洞穴 | 趣味博弈 + 中点 |
| 称霸期（成为最强） | 苦难考验 + 奖赏 + 复活 | 全盘皆输→终局 |
| 飞升（开新地图） | 携万灵丹归来 | 终场意象 |

### 扮猪吃虎 = 平坦弧 + 反差张力

| 扮猪吃虎节拍 | 叙事理论对应 |
|------------|------------|
| 主角隐藏实力 | 平坦弧（角色无缺陷，世界有偏见） |
| 被小觑 | 社会冲突（人 vs 社会偏见） |
| 被逼出手 | 价值转变（平静→爆发） |
| 震惊全场 | 期望鸿沟（读者知道主角强，但配角不知道） |
| "我只想低调" | 角色一致性（平坦弧的核心信念） |

**独特之处**：扮猪吃虎的张力来源不是"主角能不能赢"（读者知道能赢），而是"什么时候暴露"——这是悬念，不是惊奇。

### 金手指觉醒 = Propp 赠予者的前置化

| 传统叙事 | 网文改造 |
|---------|---------|
| 英雄先遇困难 → 赠予者出现提供帮助 | 赠予者在第 1 章就出现 → 英雄带着力量面对困难 |
| 赠予者是外部角色（导师/仙女/精灵） | 赠予者是系统/传承/器灵（内化为金手指） |
| 赠予是一次性的 | 金手指是持续成长的（升级解锁新功能） |

### 黄金三章 = Save the Cat 开场 10% 的极端化

| STC 开场 10% | 黄金三章对应 |
|-------------|------------|
| 开场意象（0-1%） | 第 1 章第 1 段：用一个画面定下基调 |
| 主题陈述（5%） | 第 1 章中段：暗示故事深层主题 |
| 铺垫（1-10%） | 第 1-2 章：介绍主角、世界、缺陷 |
| 催化事件（10%） | 第 3 章：打破日常的触发点 |

**黄金三章的核心**：不是"前三章要爽"，是"前三章要完成 STC 的 0-10% 节拍"——建立世界、展示缺陷、触发催化事件。

## 七、框架与追读力的关系

| 追读力类型 | 对应框架节拍 | 说明 |
|-----------|------------|------|
| 好奇心钩 | 催化事件、中点翻转 | "接下来会怎样？" |
| 爽感驱动 | 趣味博弈、终局 | "主角太帅了！" |
| 情感共鸣 | B 故事线、灵魂暗夜 | "我也经历过这种事" |
| 悬念拉力 | 全盘皆输、伏笔 | "这个谜底到底是什么？" |
| 代入感 | 铺垫、平凡世界 | "这就是我！" |
