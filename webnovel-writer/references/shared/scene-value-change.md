# 场景价值转变指南 (Scene Value Change)

> 为 plan 章纲的价值转变字段和 write 场景级质量检查提供理论工具。
> 核心来源：Robert McKee《故事》、Anton Chekhov、Alfred Hitchcock。
>
> 网文爽点工程（铺垫/兑现/微反转、压扬比例、信息差设计）详见 `cool-points-guide.md`。
> 本文件提供叙事学理论基础（价值转变、冲突类型、悬念机制），cool-points-guide 提供网文实操套路。

## 一、McKee 核心理论：每个场景必须有价值转变

Robert McKee 在《故事》(Story) 中的核心主张：**一个场景如果在结束时的核心价值状态和开始时一样，这个场景就不应该存在。**

### 什么是"价值状态"

价值状态是角色处境中最关键的情感/生存维度：

| 价值维度 | 正极 | 负极 |
|---------|------|------|
| 生存 | 安全 | 危险 |
| 信任 | 信任 | 怀疑 |
| 力量 | 强大 | 虚弱 |
| 知识 | 知晓 | 无知 |
| 孤立 | 联盟 | 孤立 |
| 道德 | 正义 | 堕落 |
| 爱情 | 相爱 | 分离 |
| 尊严 | 受敬 | 受辱 |

### 场景价值转变示例

```
场景开始：主角在宗门中安全修炼（安全+）
场景结束：发现宗门即将被灭门（危险+）

价值转变：安全 → 危险 ✅

如果场景开始和结束都是"主角在安全修炼"→ 这个场景是水字数 ❌
```

### 网文场景价值转变频率建议

| 场景类型 | 价值转变强度 | 频率 |
|---------|------------|------|
| 高潮场景 | 强转变（正极↔负极完全翻转） | 每卷 2-3 次 |
| 推进场景 | 中转变（同一维度内的微调） | 每 2-3 章 |
| 过渡场景 | 弱转变（引入新维度） | 每 5 章 |

## 二、六种冲突子类型

| 类型 | 定义 | 网文示例 |
|------|------|---------|
| **人 vs 人** | 与其他角色的对抗 | 主角 vs 反派、宗门之争 |
| **人 vs 自然** | 与自然力量的对抗 | 天劫、兽潮、秘境危险 |
| **人 vs 社会** | 与制度/规则的对抗 | 打破不公平的宗门制度 |
| **人 vs 自我** | 内心的矛盾和挣扎 | 欲望 vs 道德、复仇 vs 宽恕 |
| **人 vs 技术/命运** | 与系统/宿命的对抗 | 系统的陷阱、命运的诅咒 |
| **人 vs 未知** | 与不可理解之物的对抗 | 规则怪谈、克苏鲁、诡异事件 |

### 冲突升级策略

McKee 的"期望鸿沟"：读者对场景有一个预期，实际发生的事情打破这个预期。

```
读者预期：主角会赢得比武大会
实际发生：主角赢了，但代价是暴露了金手指
→ 期望鸿沟：胜利 + 新危机 ✅
```

升级路径：
1. 小冲突：口头争执、小摩擦
2. 中冲突：打斗、竞争、利益冲突
3. 大冲突：生死对决、阵营对立
4. 终极冲突：价值观碰撞、命运对决

## 三、悬念 vs 惊奇（Hitchcock）

Alfred Hitchcock 经典区分：

| 类型 | 定义 | 读者体验 | 效果 |
|------|------|---------|------|
| **悬念** | 读者知道危险，角色不知道 | "桌下有炸弹！快跑啊！" | 持续紧张、代入感强 |
| **惊奇** | 读者和角色都不知道 | "砰！炸弹爆炸了！" | 短暂震撼、一次性 |

**网文应用**：
- 用悬念 > 用惊奇
- 伏笔制造悬念（读者知道有危险，看主角如何应对）
- 反转制造惊奇（出乎意料但合理）
- 最佳组合：先悬念后惊奇（读者知道有炸弹，但不知道炸弹在桌下还是在墙里）

## 四、场景级微观冲突设计

### 对话级冲突

```
❌ 平淡对话：
A: "你好。"  B: "你好。"

✅ 有冲突的对话：
A: "你好。"  B: "你来干什么？"
→ 即使是寒暄，也要有态度差异
```

### 肢体/行动冲突

- 不是每场打斗都需要"打"——威胁、对峙、压迫也是冲突
- 冲突的目标不是"打赢"，是"价值转变"

### 内心冲突

- 角色在两个选项之间挣扎
- 两个选项都有代价（没有"免费的选择"）
- 选择本身推动角色弧光

## 五、价值转变检查清单

每章写完后问自己：

- [ ] 本章的核心价值维度是什么？（安全/信任/力量/知识/孤立/道德/爱情/尊严）
- [ ] 开始时的状态是什么？（正极/负极/中间）
- [ ] 结束时的状态变了没有？
- [ ] 如果没变，这章是否有存在的必要？
- [ ] 转变是通过冲突推动的，还是"自然而然"的？（冲突推动更好）

## 六、价值转变与追读力

| 价值转变方向 | 读者感受 | 追读力类型 |
|------------|---------|-----------|
| 正→负（失去） | 紧张、担心 | 悬念拉力 |
| 负→正（获得） | 爽、满足 | 爽感驱动 |
| 正→更正（升级） | 兴奋、期待 | 好奇心钩 |
| 负→更负（雪崩） | 焦虑、必须看下去 | 悬念拉力（最强） |
| 不确定→确定 | 释然 | 情感共鸣 |
